# placement

A randomized stream of product-placement clips extracted from films.

Live at **paul.tube/placement**.

## What it is

Static site. Plays an infinite shuffle of curated logo-placement clips from a manifest.

## Files

- `index.html`, `app.js`, `style.css` — the player
- `clips.json` — manifest of clips (paths, films, labels)
- `clips/` — the clip MP4s

## Updating clips

Clips are generated by the `only-the-good-stuff` pipeline. To refresh:

1. In that repo, run `python build_manifest.py` (rebuilds `web/clips.json` and copies kept clips into `web/clips/`)
2. Copy `web/*` from there to this repo
3. Commit and push — GitHub Pages will redeploy
